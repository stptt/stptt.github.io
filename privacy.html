Политика конфиденциальности приложения «Круг»

Редакция: 17 июля 2026 г.

Приложение «Круг» (далее — «Приложение») — закрытый видеомессенджер для небольших групп («кругов»): семьи и друзей. Оператор данных — владелец Приложения (контакт ниже).

1. Какие данные мы обрабатываем

Только то, что нужно для работы Приложения. Мы не собираем рекламные идентификаторы, геолокацию, контакты, историю браузера; в Приложении нет рекламы и сторонней аналитики.

Данные Зачем Где хранятся
Номер телефона вход в аккаунт (код подтверждения) база данных проекта (Supabase Auth)
Имя в профиле подпись автора видео внутри ваших кругов база данных (Supabase)
Видео, записанные в Приложении основная функция: видеосообщения вашему кругу видеоплатформа EdgeCenter (Российская Федерация): хранение, транскодирование, доставка
Реакции и отметки просмотров функции «реакции» и «кто посмотрел» база данных (Supabase)
Push-токен устройства уведомления «новое видео», «новый участник» база данных (Supabase); доставка — сервис Expo Push
Технические записи (логи серверных функций) диагностика сбоев инфраструктура Supabase, ограниченный срок

Код подтверждения входа доставляется в ваш Telegram через официальный сервис Telegram Gateway; для этого передаётся только номер телефона.

2. Кто видит ваши видео

Видео доступны только участникам круга, в который они отправлены. Это обеспечивается на уровне базы данных (политики Row-Level Security) и подписанными ссылками на видео с коротким сроком действия. Публичного доступа, поиска, рекомендаций и «обмена ссылкой наружу» нет.

3. Кому мы передаём данные

Мы не продаём и не передаём данные третьим лицам для рекламы. Данные обрабатывают только инфраструктурные подрядчики, без которых Приложение не работает:

  • EdgeCenter (ООО «ЭджЦентр», РФ) — хранение и доставка видео;
  • Supabase — база данных и аутентификация;
  • Telegram Gateway — доставка кода входа;
  • Expo (Exponent) — доставка push-уведомлений.

4. Сроки хранения и удаление

  • Видео — автор может удалить своё видео в Приложении в любой момент; удаление убирает его из ленты круга и с видеоплатформы.
  • Выход из круга — доступ к его видео прекращается.
  • Удаление аккаунта и всех данных — прямо в Приложении: «Настройки» → «Удалить аккаунт». Аккаунт, профиль, ваши видео, реакции, отметки просмотров и push-токены удаляются безвозвратно. Если у вас больше нет доступа к Приложению, пришлите запрос на контактный адрес ниже. Подробнее — на странице «Удаление аккаунта».

5. Безопасность

Данные передаются по защищённым соединениям (HTTPS/TLS). Доступ к видео — только по подписанным ссылкам с ограниченным сроком действия. Серверные ключи недоступны из Приложения.

6. Дети

Приложение не предназначено для детей младше 13 лет. Круги закрыты: попасть в круг можно только по приглашению его участника. Наши правила в отношении защиты детей и противодействия CSAE — на странице «Стандарты безопасности детей».

7. Изменения политики

Актуальная редакция публикуется по этому же адресу; существенные изменения анонсируются в Приложении.
